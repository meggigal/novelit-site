노벨릿 웹소설 표지 패키지

작업 스타일과 예산에 맞춰, 세 가지 패키지 중 선택하실 수 있습니다.

라이트

가볍게 시작하는 웹소설 표지

  • 상반신 캐릭터 1명 + 최소한의 리터칭
  • 타이포 X
  • 시안 1개 / 수정 1회

스탠다드

웹소설 플랫폼 최적화 표지

  • 캐릭터 1명 + 얼굴/손/색감/배경 리터칭
  • 타이포 O
  • 시안 2개 / 수정 2회

프리미엄

세계관을 한 장에 담는 커스텀 표지

  • 캐릭터 최대 3명 + 스탠다드 리터칭 + 디테일 강화
  • 배경 디테일 + 이펙트
  • 시안 4개 / 수정 3회

라이트 패키지

가볍게 시작하는 표지

구성

  • 상반신 중심 캐릭터 1명
  • AI 일러스트 베이스 + 기본 색보정
  • 타이포 X (제목/작가명은 직접 넣는 분에게 추천)
  • 시안 1개 제공
  • 수정 1회 제공 (색감, 표정, 소소한 디테일 위주)

이런 분께 추천

  • 표지를 빠르게 맞추고 싶은 연재 초반 작가
  • 이미 타이포를 직접 넣거나 디자이너가 따로 있는 경우
  • 저렴하게 일러스트만 필요한 경우

스탠다드 패키지

웹소설 플랫폼 최적화 표지

구성

  • 상반신~반신 위주 캐릭터 1명
  • 다양한 구도 시도 (예: 정면, 3/4, 약간의 원근)
  • AI 일러스트 + 기본 리터칭(얼굴, 손, 디테일 정리)
  • 타이포 O (제목 / 부제 / 작가명 레이아웃 포함)
  • 기본 색보정 + 콘트라스트 튜닝
  • 시안 2개 제공 / 수정 2회 제공

이런 분께 추천

  • 카카오페이지, 리디, 네이버 시리즈 등 플랫폼에 맞는 표지를 원할 때
  • “그냥 일러”가 아니라 어느 정도 완성된 홍보용 이미지를 원할 때
  • 장르 정체성(로판/현판/학원 등)을 확실히 보여주고 싶은 경우

프리미엄 패키지

세계관을 한 장에 담는 커스텀 표지

구성

  • 인물 최대 3명까지 추가 가능 (주인공 + 라이벌/히로인 등 조합)
  • 배경 디테일 강화 (도시/궁전/학원/던전 등 세계관 표현)
  • 풍부한 이펙트 추가 (마법, 파티클, 번쩍이는 빛, 스킬 연출 등)
  • 리터칭 많이 (얼굴/손/의상 주름/장식 등 섬세한 보정)
  • 타이포 O – 커스텀 느낌 (제목 스타일, 서체 조합, 레이아웃 신경 써서 구성)
  • 시안 최대 4개 제공 / 수정 총 3회 제공

이런 분께 추천

  • 메인 홍보배너/광고용 이미지까지 염두에 두는 작가
  • 시리즈물, 시즌제, 유료연재 중심 작품
  • 캐릭터 관계와 세계관을 한 장에 강하게 보여주고 싶은 경우
